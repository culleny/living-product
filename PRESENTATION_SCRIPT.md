## Q&A Preparation

**Q: How does it handle different product types?**
A: The campaign brief adapts the background and style. We support both 2D images and 3D models.

**Q: What if an API call fails?**
A: Each step has fallbacks. Video is optional, Photoshop falls back to Firefly. Pipeline always completes.

**Q: Can it handle batch processing?**
A: Current version processes one at a time, but architecture supports parallel processing. That's Phase 2.

**Q: How much does it cost to run?**
A: Minimal - S3 storage is pennies, Firefly API calls are per-image. Much cheaper than designer hours.

**Q: Is it production-ready?**
A: Yes! We've tested end-to-end with real AWS and Adobe credentials. Code includes error handling, logging, and retry logic.

**Q: What formats does it output?**
A: 8 formats: hero render, cutout, branded hero, billboard (1920x1080), video, story (9:16), square (1:1), banner (16:9).
